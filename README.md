# User Login Portal 🚀

A professional and clean **User Login & Registration System** built using HTML, CSS, and JavaScript. This project features a modern dark-themed UI and local data management.

## ✨ Features
* **Modern UI:** Dark mode design with smooth transitions.
* **Dual Mode:** Easily switch between Login and Registration forms.
* **Local Storage:** Saves user data directly in the browser (no backend required for testing).
* **Admin Panel:** Access a secret list of all registered users.
* **Responsive:** Works well on different screen sizes.

## 🛠️ Tech Stack
* **HTML5:** Semantic structure.
* **CSS3:** Custom styling and Poppins font integration.
* **JavaScript (ES6):** Logic for authentication and data storage.

## 🔑 How to Use the Admin Panel
To see the list of all registered users:
1. Go to the Login page.
2. Enter Username: `admin`
3. Enter Password: `showme`
4. The user list table will appear automatically!

## 🚀 How to Run
1. Clone or download this repository.
2. Open `index.html` in any modern web browser.
